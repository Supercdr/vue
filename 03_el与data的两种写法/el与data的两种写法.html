el的两种写法

    容器与Vue实例的绑定方法

  • 在实例里面使用el方法绑定

    例如 

              new Vue({
            el:'#root', 
            data:{
              name:'kaylee'
            }
          })

  • 继承vue的内部方法mount将Vue实例挂载到容器,这个方法更加灵活
    先创建一个vue实例，然后在后面使用 x.$mount('#root') 方法进行绑定

    例如： 

              const x=new Vue({
            data:{
              name:'kaylee'
            }
          })
          console.log(x)
          x.$mount('#root') //继承vue原型的mount方法将vue实例x挂载在id为root的容器上

data的两种写法

    Vue实例中data的两种写法（后面必须使用函数）

  • 创建对象

    例如： 

              data:{
            name:'kaylee'
          }

  • 函数（不能使用箭头函数，否则函数中的this对象是window而不是Vue实例）

    例如： 

              data:function(){
            return{   //返回一个对象
              name:'kaylee'
            }
          }
